Приложение MemoryTraining.
Приложение предназначено для тренировки памяти. Написано на языке python с использованием библиотеки PyQt5.
Была реализована база данных, в которой сохраняются логин, пароль, лучший результат. Лучший результат представляет из себя две переменные: затраченное время в секундах и кол-во ошибок.
Результат высчитывается по формуле: 1000 - time - mistakes * 5, где time - затраченное время в секундах, mistakes - кол-во ошибок.
Реализована функция смены пароля.
При регистрации есть проверка на то, чтобы пароль был длиннее 5 символов. При смене пароля эта проверка тоже присутствует.
Игра представляет из себя набор картинок 4*4 рандомно разбросанных. При каждой новой игре происходит случайное перемешивание картинок. Первые 5 секунд карточки открыты, чтобы запомнить.
В игре реализована функция паузы. При паузе все картинки переворачиваются и время останавливается. 
Если выйти из игры, не завершив ее, то результат не сохранится.
После нахождения всех одинаковых пар появится окно с вашим результатом.
Лучший результат можно посмотреть в главном меню при нажатии кнопки rating.
